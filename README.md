# PrismLauncher-without-MS-account
a modified accounts.json file to allow user to play MC without Microsoft account

# Instruction
* download and install PrimsLauncher like usual
* download and move/replace accounts.json file from this repo to PrimsLauncher folder (might have to replace old file, be aware to backup your account list if you have them before)
* launch the game, you should see FakeAuthMCAccount added to account list, now you can start playing like normal
